Auto Appreciations 

Preparation du fichier

Sur Pronote, copier le tableau des notes en format .csv puis l'ouvrir avec Excel, copier la partie nom des eleves
et leurs notes, supprimer les moyennes en fin de tableau
Copier les valeurs dans le fichier Excel d'exemple: Appreciations_notes.xlsx

Requierements

Python 3.11:
telecharger depuis le site officiel et cocher la case " Add Python to PATH" lors de l'installation 


Librairies:Ouvrir l'invité de commandes (cmd) de Windows en mode administrateur puis installer les librairies
en exécutant les lignes de commandes citées

 pandas:
pip install pandas

 openpyxl:
python -m pip install openpyxl

Instructions:
Copier les données élèves dans le fichier Excel
Lancer AutoApp 
Choisir le fichier Excel
Cliquer sur Sauvegarder en fichier texte
Ouvrir le fichier Texte créé puis copier le texte et l'insérer dans n'importe quelle IA
Et voilà !
